Kraken Ledgers Help

Uploading Kraken Ledgers

  • Go to Sources => Kraken Ledgers
  • Via Browse, select the csv that contains your Kraken ledgers
  • Click on Upload

    If you see a message showing there are new Kraken assets to be refreshed, check the log in the main window for more details on the missing assets. The asset code(s) should appear in Kraken Assets, which can be reached via Configure => Kraken Assets.

    • If you do not see the asset code(s), try to Refresh from source.

    • Fill in the missing asset part for the asset code(s). The asset should match an asset entry in Asset Catalog, which can be reached via Configure => Asset Catalog.
