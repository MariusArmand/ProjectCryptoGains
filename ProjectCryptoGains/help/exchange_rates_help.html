Exchange Rates Help

What are Exchange Rates?

Exchange rates in PCG are stored per DATE, ASSET, FIAT_CURRENCY combination; Every time an exchange rate is needed and it is not yet stored by PCG, PCG will retrieve the exchange rate via the CoinDesk Data API and store it for later use Exchange rates from the CoinDesk Data API are retrieved using the opening price on the relevant date; This approach limits API calls and maintains performance, with the assumption that consistently using opening prices provides a balanced representation of historical price data

Import csv requirements

Layout

The exchange rates csv should have columns seperated with a , and use . as decimal seperator

Columns

date The date of the exchange rate in YYYY-MM-DD format
asset The asset code
fiat_currency The fiat currency to which the asset is converted
exchange_rate The price of the asset in the fiat currency on the date

Example

"date","asset","fiat_currency","exchange_rate"
"2020-11-13","LTC","EUR",51.54
"2020-11-13","ETH","EUR",392.37
"2021-01-02","XMR","EUR",111.16
"2021-01-02","BTC","EUR",24028.41

Importing Exchange Rates

  • Go to Sources => Exchange Rates and select the csv that contains the exchange rates
  • Click on Import
