Gains Help

Gains are calculated from the data produced in the Process => Trades, using the LIFO (Last In, First Out) method
REFID Reference ID
DATE Date in YYYY-MM-DD HH:MI:SS format (in UTC time zone)
TYPE Can be "BUY" or "SELL"
BASE_CURRENCY Base asset Class
BASE_AMOUNT Amount debited or credited to the base asset's balance
QUOTE_CURRENCY Quote asset Class
QUOTE_AMOUNT Amount debited or credited to the quote asset's balance
BASE_UNIT_PRICE_FIAT The cost of one unit of the base asset in terms of the quote asset, converted to fiat
COSTS_PROCEEDS The costs (in case of "BUY") or proceeds (in case of "SELL")
TX_BALANCE_REMAINING The buy transaction's base asset balance remaining after the LIFO processing
GAIN

The gain of the sell transaction calculated via the LIFO method Calculation (simplified): sell.PROCEEDS - (sell.AMOUNT * previous buy.BASE_UNIT_PRICE_FIAT)
