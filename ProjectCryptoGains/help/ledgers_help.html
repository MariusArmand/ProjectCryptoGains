Ledgers Help

Ledgers are generated using the data uploaded via Sources
REFID Reference ID
DATE Date in YYYY-MM-DD HH:MI:SS format (in UTC time zone)
TYPE Can be DEPOSIT, WITHDRAWAL, TRADE, EARN, STAKING
EXCHANGE Exchange involved
CURRENCY Asset
AMOUNT Amount debited (-) or credited (+)
FEE Fee paid (if any) in the asset
SOURCE Origin of transaction
TARGET Goal of transaction
NOTES Short transaction description
