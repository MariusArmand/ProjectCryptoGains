Rewards Help

Rewards are generated using the data generated in Process => Ledgers
REFID Reference ID
DATE Date in YYYY-MM-DD HH:MI:SS format (in UTC time zone)
TYPE Can be "EARN" or "STAKING"
EXCHANGE Exchange involved
CURRENCY Currency code
AMOUNT Reward amount
AMOUNT_FIAT

Reward amount converted to fiat Calculation: AMOUNT * UNIT_PRICE
TAX

Tax calculated using Rewards Tax Percentage, which can be set at Configure => Settings Calculation: AMOUNT_FIAT * Rewards Tax Percentage
UNIT_PRICE The fiat price of one unit of the asset
UNIT_PRICE_BREAK_EVEN

The approximate price per unit at which selling the asset would cover the tax cost while maintaining the value of the original reward amount in fiat Calculation: UNIT_PRICE + (UNIT_PRICE * TAX)
AMOUNT_SELL_BREAK_EVEN

The amount of asset units to sell at the UNIT_PRICE_BREAK_EVEN to approximately cover the tax obligation, with the goal of the remaining amount retaining the original reward's value in fiat Calculation: TAX / UNIT_PRICE_BREAK_EVEN

Summary

CURRENCY Currency code
AMOUNT Reward amount
AMOUNT_FIAT Sum of AMOUNT_FIAT
TAX Sum of TAX
UNIT_PRICE* Average UNIT_PRICE
UNIT_PRICE_BREAK_EVEN* Average UNIT_PRICE_BREAK_EVEN
AMOUNT_SELL_BREAK_EVEN* Sum of AMOUNT_SELL_BREAK_EVEN

*Rough estimates based on averages and sums, not exact figures
