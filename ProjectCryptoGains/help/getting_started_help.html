Getting Started

1. Configure Settings

Go to Configure => Settings

Fiat currency

Select the fiat currency you wish to use for conversions in Project Crypto Gains (PCG)

CryptoCompare API key

PCG uses the CCData/CryptoCompare API for conversions to fiat; Go to the CCData website to sign up for an API key that suits your needs

2. Import source data

Go to
  • Sources => Manual Ledgers*
  • Sources => Kraken Ledgers

*optional

Each import window has its own help section; Assets unknown to PCG will need to be configured via
  • Configure => Asset Catalog
  • Configure => Kraken Assets

3. Process

3.1 Refresh Ledgers

Go to Process => Ledgers to refresh PCG ledgers

3.2 Refresh Trades

Go to Process => Trades to refresh PCG trades

- PCG uses ledgers and trades as input for generating other reports (e.g., Gains, Rewards, etc.) - After importing new source data, always refresh the PCG ledgers and trades (in that order), before refreshing any other reports..
