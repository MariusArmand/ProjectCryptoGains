Trades Help

Trades are generated using the data generated in Process => Ledgers
REFID Reference ID
DATE Date in YYYY-MM-DD HH:MI:SS format (in UTC time zone)
TYPE Can be "BUY" or "SELL"
EXCHANGE Exchange involved
BASE_ASSET Base asset code
BASE_AMOUNT Amount of base asset
BASE_FEE Fee paid (if any) in the base asset
BASE_FEE_FIAT BASE_FEE converted to fiat
QUOTE_ASSET Quote asset code
QUOTE_AMOUNT Amount of quote asset
QUOTE_AMOUNT_FIAT QUOTE_AMOUNT converted to fiat
QUOTE_FEE Fee paid (if any) in the quote asset
QUOTE_FEE_FIAT QUOTE_FEE converted to fiat
BASE_UNIT_PRICE

The cost of one unit of the base asset in terms of the quote asset Calculation: QUOTE_AMOUNT / BASE_AMOUNT
BASE_UNIT_PRICE_FIAT BASE_UNIT_PRICE converted to fiat
QUOTE_UNIT_PRICE

The cost of one unit of the quote asset in terms of the base asset Calculation: BASE_AMOUNT / QUOTE_AMOUNT
QUOTE_UNIT_PRICE_FIAT QUOTE_UNIT_PRICE converted to fiat
COSTS_PROCEEDS

The costs (in case of "BUY") or proceeds (in case of "SELL") Calculation: -COSTS: ABS(QUOTE_AMOUNT) + BASE_FEE_FIAT + QUOTE_FEE_FIAT -PROCEEDS: ABS(QUOTE_AMOUNT) - BASE_FEE_FIAT - QUOTE_FEE_FIAT
