Asset Catalog Help

What is the Asset Catalog?

The Asset Catalog assigns a descriptive label to each asset All report processing in PCG relies on the assets defined within the Asset Catalog

How to maintain the Asset Catalog

Columns

ASSET Asset code (e.g., BTC) used throughout PCG; Should be unique
LABEL

- Descriptive label for the asset (e.g., Bitcoin); Should be unique - This should match with LABEL in Kraken Assets - Appears in parentheses in summary reports and the Balances report, paired with the ASSET column (e.g., BTC (Bitcoin))
