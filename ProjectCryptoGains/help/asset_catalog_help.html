Asset Catalog Help

What is the Asset Catalog?

The Asset Catalog contains per asset CODE, a more descriptive name (ASSET) to be used everywhere in PCG.

How to maintain the Asset Catalog

Columns

CODE Code for asset (e.g. BTC). Should be unique.
ASSET

More descriptive name for the asset (e.g. BTC (Bitcoin)). Should be unique. This should match with ASSET in Kraken Assets and LEFT_ASSET and RIGHT_ASSET in Kraken Pairs.
