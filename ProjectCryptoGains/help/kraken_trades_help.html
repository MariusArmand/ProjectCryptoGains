Kraken Trades Help

Uploading Kraken Trades

  • Go to Sources => Kraken Trades
  • Via Browse, select the csv that contains your Kraken trades
  • Click on Upload

    If you see a message showing there are new Kraken pair codes to be refreshed, check the log in the main window for more details on the missing pairs. The pair code(s) should appear in Kraken Pairs, which can be reached via Configure => Kraken Pairs.

    • If you do not see the pair code(s), try to Refresh from source.

    • Fill in the missing asset parts (left/right) for the asset code(s). The asset parts (left/right) should match an asset entry in Asset Catalog, which can be reached via Configure => Asset Catalog.
