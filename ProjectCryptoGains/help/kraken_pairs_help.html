Kraken Pairs Help

What is Kraken Pairs?

Kraken Pairs contains the distinct pairs (CODE) that are uploaded by Kraken Trades.

How to maintain the Asset Catalog

Columns

CODE Code for pair (e.g. LTC/ETH). Should be unique.
ASSET_LEFT

More descriptive name for the left asset of the pair (e.g. LTC (Litecoin)). This should match with ASSET in Asset Catalog.
ASSET_RIGHT

More descriptive name for the right asset of the pair (e.g. ETH (Ethereum)). This should match with ASSET in Asset Catalog.

Troubleshooting

Upon saving, if you see a message showing there were malconfigured pair(s) detected, check the log in the main window for more details on the malconfigured pair(s). Doublecheck that ASSET_LEFT and ASSET_RIGHT match their corresponding ASSET in the Asset Catalog, which can be reached via Configure => Asset Catalog. If there is no typo and there is no corresponding ASSET in the Asset Catalog, you need to add the ASSET in the Asset Catalog.
