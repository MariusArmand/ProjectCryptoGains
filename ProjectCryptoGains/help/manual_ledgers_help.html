What is Manual Ledgers?

Manual Ledgers enables you to let PCG take transactions into account that were not done via Kraken.

How to create Manual Ledgers

Layout

The manual ledgers csv should have columns seperated with a , and use . as decimal seperator.

Columns

refid

This can be anything as long as it is unique for the transaction. All types of transactions take 1 ledger entry, except TRADE, which takes 2 ledger entries.
The 2 ledger entries for the TRADE transaction should have the same refid.

date

The date of the transaction in YYYY-MM-DD HH:MI:SS format.

type

The transaction type. Supported types are DEPOSIT, WITHDRAWAL, TRADE, STAKING

exchange*

The exchange used in the transaction if applicable

asset

The transaction asset. This should match with an ASSET from the Asset Catalog. The Asset Catalog can be found via Configure => Asset Catalog.

amount

The transaction amount without the fee.

fee

The transaction fee.

source*

The source of the transaction.

target*

The target of the transaction.

notes*

Notes to give additional context to the transaction.

*optional

Example

refid,date,type,exchange,asset,amount,fee,source,target,notes
MANUAL-0,2018-02-27 08:10:00,DEPOSIT,Coinbase,EUR,1000,0,BANK,Coinbase,From Bank to Coinbase
MANUAL-1,2018-02-27 15:19:00,TRADE,Coinbase,BTC (Bitcoin),0.1165775,0,,,
MANUAL-1,2018-02-27 15:19:00,TRADE,Coinbase,EUR,-985.32,14.68,,,
MANUAL-2,2018-03-20 10:45:00,WITHDRAWAL,Coinbase,BTC (Bitcoin),0.1165775,0.00176159,Coinbase,WALLET,From Coinbase to wallet
MANUAL-3,2024-12-31 23:59:59,STAKING,,TAO (Bittensor),0.003,0,WALLET,WALLET,TAO (Bittensor) staking reward

Uploading Manual Ledgers

  • Go to Sources => Manual Ledgers
  • Via Browse, select the csv that contains your manual ledgers
  • Click on Upload

    • If you see a message that there were missing assets detected, check the log in the main window for more details on the missing assets.
    The transaction asset should match the Asset Catalog, which can be reached via Configure => Asset Catalog.
    If the asset is indeed missing in the Asset Catalog, it can be added and saved in that window.
