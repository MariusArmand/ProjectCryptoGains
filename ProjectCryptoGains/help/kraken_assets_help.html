Kraken Assets Help

What is Kraken Assets?

Kraken Assets contains the assets (CODE) that are uploaded by Kraken Ledgers

How to maintain the Asset Catalog

Columns

CODE Code for asset (e.g. BTC)

It's possible to have similar CODEs (e.g. BTC, BTC.M) where BTC.M is the rewards code for BTC; In that case, you should set the same ASSET for both CODEs (e.g. BTC (Bitcoin))
ASSET

- More descriptive name for the asset (e.g. BTC (Bitcoin)) - This should match with ASSET in Asset Catalog

Troubleshooting

- Upon saving, if you see a message showing there were malconfigured asset(s) detected, check the log in the main window for more details on the malconfigured asset(s) - Doublecheck that ASSET matches their corresponding ASSET in the Asset Catalog, which can be reached via Configure => Asset Catalog - If there is no typo and there is no corresponding ASSET in the Asset Catalog, you need to add the ASSET in the Asset Catalog
