Kraken Assets Help

What is Kraken Assets?

Kraken Assets mangages the assets that are uploaded by Kraken Ledgers

How to maintain the Asset Catalog

Columns

ASSET Asset code (e.g., BTC)

It's possible to have similar ASSETs (e.g., BTC, BTC.M) where BTC.M is the rewards asset for BTC; In that case, you should set the same LABEL for both ASSETs (e.g., Bitcoin)
LABEL

- Descriptive label for the asset (e.g., Bitcoin) - This should match with LABEL in Asset Catalog

Troubleshooting

- Upon saving, if you see a message showing there were malconfigured asset(s) detected, check the log in the main window for more details on the malconfigured asset(s) - Doublecheck that LABEL matches their corresponding LABEL in the Asset Catalog, which can be reached via Configure => Asset Catalog - If there is no typo and there is no matching LABEL in the Asset Catalog, you need to add the ASSET/LABEL in the Asset Catalog
